Our pacmanAI is implemented using depth-first search and breadth-first search.
To run it you have to complete next steps:


1. Go to the directory where you want to save the project and run the command:

 $ git clone https://github.com/vrshchk/pacmanAI


2. Install neccessery libraries:

 $ pip install pygame
 $ pip install random
 $ pip install sys
 $ pip install time


3. Go to the downloaded directory and run the command:

 $ python main.py
